

### 프로젝트 소개

개인적으로 진행하는 프로젝트 이며, 학교 졸업생 커뮤니티 웹사이트 개발이 주된 목표입니다.


### 주요 기능

User : 회원가입, 비밀번호 변경, SMTP, 로그인, 로그아웃

Community : 게시글 작성




### 기술스택

FrontEnd : Html/Css/Javascript, BootStrap4

BackEnd : Django(Python - anaconda)

(Django = 3.2.4), (python = 3.9.12), (conda = 4.12.0)
